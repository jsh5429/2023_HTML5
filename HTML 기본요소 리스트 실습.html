테이블태그는 tr로 열을 작성하고 td로 안에 셀을 만든다.
tr : table row(열), td : table data(셀),
th : table header(h1태그처럼 생각하면 된다.)
colspan : 오른쪽에 있는 셀을 확장
rowspan : 아래쪽에 있는 셀을 확장
