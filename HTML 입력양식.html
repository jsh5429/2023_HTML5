양식 중식 한식

radio타입의 경우 name을 통해서 그룹으로 묶여진다.
그룹으로 묶여졌을 경우 그룹 중에 하나만 선택할 수 있다.
input 태그의 value 속성은 서버로 보낼 값이 된다.
text나 password처럼 값을 입력받는 경우가 아니라면 미리 적어둔다.

체크박스


HTML CSS JAVA C++

체크박스는 여러개를 선택할 때 사용할 수 있는 type이다.
checked, disabled 속성으로 체크박스의 상태를 지정할 수 있다.
라디오버튼도 동일하게 적용된다.
